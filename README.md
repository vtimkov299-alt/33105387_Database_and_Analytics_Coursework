# NorthStar Logistics Data Analysis

## Overview
This project analyses a logistics dataset using Python, R (SQL-style queries), and MongoDB. The aim is to investigate delivery performance, customer complaints, and operational efficiency across different zones and service types.

## Technologies Used
- Python (pandas, matplotlib, pymongo)
- R (dplyr, sqldf, ggplot2)
- MongoDB Atlas (NoSQL database)
- SQL-style queries within R

## Key Features
- Data cleaning and preprocessing (Python)
- Exploratory and statistical analysis (Python & R)
- Data visualisation using matplotlib and ggplot2
- MongoDB data storage, querying, and aggregation
- Query optimisation using indexing

## Dataset
The dataset includes:
- Orders
- Deliveries
- Complaints
- Drivers
- Vehicles
- Incidents

## How to Run
1. Open the notebook in Google Colab
2. Run all cells step-by-step
3. Ensure required libraries are installed
